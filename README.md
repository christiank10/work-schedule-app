# 📅 Minha Escala Pro - Gerador de Espelho de Ponto

Aplicação web intuitiva desenvolvida para facilitar o controle de escalas de trabalho e geração de relatórios em PDF. Ideal para trabalhadores que precisam organizar seus horários de entrada, almoço e saída de forma rápida e profissional.

## 🚀 Funcionalidades
- **Preenchimento Dinâmico:** Interface em cards para fácil visualização dos dias do mês.
- **Armazenamento Local:** Os dados são salvos no navegador (LocalStorage), garantindo que nada se perca ao fechar a página.
- **Exportação para PDF:** Gera um espelho de ponto formatado em folha A4, pronto para impressão.
- **Totalizador:** Cálculo automático de dias trabalhados no mês.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Custom Styles para A4) e Bootstrap 5.
- **Lógica:** JavaScript Vanilla.
- **Biblioteca de PDF:** [html2pdf.js](https://ekoopmans.github.io/html2pdf.js/).

## 📄 Como usar
1. Selecione o mês e o ano desejado.
2. Clique no card do dia para inserir os horários.
3. Após preencher o mês, clique no botão **PDF** para baixar o relatório ajustado.
